📝 项目介绍

本项目是一款基于 HTML5、CSS3 和原生 JavaScript 开发的完整扫雷游戏应用。 采用模块化架构设计，包含游戏核心引擎、本地数据持久化、多主题切换、 排行榜与成就系统、触屏手势支持等功能，致力于在浏览器端提供流畅、 沉浸的扫雷游戏体验。

💡 技术栈

HTML5 CSS3 JavaScript (ES6+) LocalStorage Web Audio API Responsive Design

⚙️ 核心功能

  • 三级难度自由选择（初级 / 中级 / 高级）
  • 首次点击保护机制，确保开局安全
  • 智能连锁翻开空白区域
  • 双击数字格快捷翻开周围
  • 暂停/继续功能
  • 本地排行榜与成就系统
  • 三种主题皮肤切换
  • 完整的触屏手势支持
  • 游戏成绩一键分享

📖 操作指南

🎮 基础操作

  • 左键单击：翻开格子，揭示数字或空白区域
  • 右键单击：在格子上插旗标记 suspected 地雷
  • 双击数字格：当周围插旗数等于该数字时，自动翻开剩余邻居

📱 触屏操作

  • 短按：翻开格子（等同于左键）
  • 长按 0.5 秒：标记/取消标记（等同于右键）

⌨️ 键盘快捷键

  • R：重新开始当前游戏
  • 1 / 2 / 3：快速切换初级/中级/高级
  • ESC：暂停/继续游戏

🌟 游戏目标

翻开所有非地雷格子即可获胜。数字表示周围 8 格中地雷的数量。善用标记和逻辑推理，避开所有地雷！
